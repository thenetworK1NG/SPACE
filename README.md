# Endless Space — Three.js Demo

Simple endless-space demo using Three.js. It uses Pointer Lock + WASD for free flight and expects a `ship.glb` model placed at the project root.

Run: serve the folder with any static server (or open `index.html` via a dev server).

Quick start (recommended):

1. Place your `ship.glb` at the project root (next to `index.html`).
2. Start a simple static server (example with Python 3):

```powershell
python -m http.server 8080
```

3. Open http://localhost:8080 in your browser, click the start button, then use WASD + mouse.
